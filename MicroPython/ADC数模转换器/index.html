  • ADC数码转换器介绍

模拟信号（Analog Signal）：模拟信号是连续变化的量或者信号，生活中接触到的信号基本都是模拟信号，温度变化，天体运动等等，这些都是连续的信息，都是模拟信号。模拟信号，简单的说就是用电信号模拟出其他的信号，比如用电信号模拟出图像，模拟出声音的声波。

数字信号（Digital Signal）：数字信号是时间离散、数值离散的信号，数字信号存在采样，还存在量化，只能取到一些不连续的固定值，这也是数字信号和模拟信号之间可以进行相互转换的原因。

所以，总结就是模拟信号时间连续，幅值连续，数字信号时间离散，幅值离散。模拟电路就是使用、处理模拟信号的电路；数字电路就是使用、处理数字信号的电路。

ADC（Analog to Digital Converter）即模数转换器，它可以将模拟信号转换为数字信号。由于单片机只能识别二进制数字，所以外界模拟信号常常会通过 ADC 转换成其可以识别的数字信号。常见的应用就是将变化的电压转成数字信号。

  • ADC的类和函数

构造方法 machine.ADC(pin, atten)

参数 pin：Pin 对象，

参数 atten：配置衰减器增加电压测量范围。

设置衰减（atten(atten)）
ADC默认只能输入0~1v的电压，要测量其他范围的电压可以对输入电压进行衰减， 可以通过设置atten值进行衰减，atten的衰减值有;

  1. ATTN_0DB：测量电压范围： 0-1.0V（默认值）

  2. ATTN_2_5DB：测量电压范围： 0-1.34V`

  3. ATTN_6DB：测量电压范围：0-2.0V`

  4. ATTN_11DB：测量电压范围：0-3.3V 这是最常用的值

设置分辨率（width(bit)）
ADC 分辨率越高，反应的模拟电压值就越精确。bit的精度值有：

  1. ADC.WIDTH_9BIT - ADC值范围0~511

  2. ADC.WIDTH_10BIT - ADC值范围0~1023

  3. `ADC.WIDTH_11BIT`` - ADC值范围0~2047

  4. `ADC.WIDTH_12BIT`` - ADC值范围0~4095（默认为12BIT）

读取ADC值（read()）

返回指定分辨率范围的模拟电压转换成的ADC值，默认分辨率为12BIT，是的范围就是0~4095。

  1. read_uv()返回毫伏单位，范围0~4095000毫伏的值

  2. read_u16()返回0~65535的值

实验二：使用旋钮编码器控制LED灯的亮度

程序如下：

import time
from machine import Pin, ADC,PWM

# 在 32 引脚创建 ADC 对象
adc = ADC(Pin(32, Pin.IN),atten=ADC.ATTN_11DB)
led = PWM(Pin(2, Pin.OUT), freq=1000)

while True:
    '''
    ADC的读取值范围是0~4096；
    PWM的占空比duty占空比范围是0~1023;
    PWM的占空比duty_u16占空比范围是0~4096,所以u16的占空比额ADC的值完美映射
    '''
    led.duty_u16(adc.read_u16())
    time.sleep(0.1)

实验三：使用旋钮编码器控制LED灯的亮度（自定义映射的方式）

对应这个案例中ADC的值范围和PWM的U16占空比范围刚好都是0~4096，所以不需要比率映射，如果是两个不同的范围呢比如使用PWM的duty（）他的范围是0~1023，那该如何做呢？这里我们可以自己设计一个映射函数就可以完美解决了。

程序如下

import time
from machine import Pin, ADC,PWM

#  定义ADC对象和PWM对象
adc = ADC(Pin(32, Pin.IN),atten=ADC.ATTN_11DB)
led = PWM(Pin(2, Pin.OUT), freq=1000)


# 自定义一个映射函数
def math_map(x, in_min, in_max, out_min, out_max):
    """
    将 x 从 [in_min, in_max] 线性映射到 [out_min, out_max]
    """
    if in_min == in_max:
        return (out_min + out_max) / 2  # 返回中间值(防止分母出现0的除零bug)
    else:
        mapped = (x - in_min) * (out_max - out_min) / (in_max - in_min) + out_min
        return max(out_min, min(mapped, out_max))  # 约束在输出范围内

while True:
    value = adc.read()
    duty = int(math_map(value,0,4095,0,1023))
    led.duty(duty)
    time.sleep(0.1)

实验四：使用旋钮编码器控制LED灯的亮度（调用自建函数）

我们可以把前面案例中的映射函数保存到common的文件夹中，文件名就是math_map，以后用到的时候直接调用。

import time
from machine import Pin, ADC,PWM
from  common.math_map import math_map # 导入刚自建的函数

#  定义ADC对象和PWM对象
adc = ADC(Pin(32, Pin.IN),atten=ADC.ATTN_11DB)
led = PWM(Pin(2, Pin.OUT), freq=1000)

while True:
    value = adc.read()
    duty = int(math_map(value,0,4095,0,1023))
    led.duty(duty)
    time.sleep(0.1)
