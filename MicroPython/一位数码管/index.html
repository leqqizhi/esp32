一位数码管

LED数码管是将发光二极管做成段，然后把段组合成“8”的形状，控制不同段的发光组合，形成数字样式。

一位数码管是一种常见的显示器件，由 7 个 LED 段（a-g） 和 1 个小数点（dp） 组成，通过不同段的组合显示数字（0-9）或部分字母（如 A-F）。其名称“一位”表示它只能显示单个字符。

因直接使用数码管，需要连接很多的接线，比较繁琐，现在常见的数码管都使用了 74HC595串行输入/并行输出移位寄存器芯片，这样就大大减少了引脚的占用，而且目前有成熟的第三方库文件可以使用，大大的降低了编程的难度

  • 第三方库

我们在libs文件夹中新建一个SegmentDisp.py文件，把网上的地方放库保存进去就可以使用了：

from machine import Pin
import time

class SegmentDisplay:
    def __init__(self, data_pin, clock_pin, latch_pin, common_cathode=True):
        """
        初始化 74HC595 数码管驱动类
        :param data_pin:  数据引脚 (DS)
        :param clock_pin: 时钟引脚 (SHCP)
        :param latch_pin: 锁存引脚 (STCP)
        :param common_cathode: 数码管类型 (True为共阴，False为共阳)
        """
        self.data = Pin(data_pin, Pin.OUT)
        self.clock = Pin(clock_pin, Pin.OUT)
        self.latch = Pin(latch_pin, Pin.OUT)
        self.common_cathode = common_cathode

        # 定义共阴数码管段码表 (0-9, A-F, 小数点)
        self.segment_codes = {
            '0': 0x3F, '1': 0x06, '2': 0x5B, '3': 0x4F,
            '4': 0x66, '5': 0x6D, '6': 0x7D, '7': 0x07,
            '8': 0x7F, '9': 0x6F, 'A': 0x77, 'B': 0x7C,
            'C': 0x39, 'D': 0x5E, 'E': 0x79, 'F': 0x71,
            '.': 0x80, ' ': 0x00, '5.': 0x6D | 0x80   # 注意小数点的显示方法
        }

        # 如果是共阳数码管，段码取反
        if not self.common_cathode:
            for key in self.segment_codes:
                self.segment_codes[key] ^= 0xFF

    def _shift_out(self, value):
        """向74HC595移位寄存器发送一个字节数据"""
        for _ in range(8):
            self.data.value((value >> 7) & 0x01)  # 发送最高位
            self.clock.value(1)
            time.sleep_us(1)
            self.clock.value(0)
            time.sleep_us(1)
            value <<= 1  # 左移准备发送下一位

    def display(self, char):
        """
        显示指定字符
        :param char: 要显示的字符 (0-9, A-F, 或小数点 '.')
        """
        # 获取段码，默认显示空格（全灭）
        code = self.segment_codes.get(str(char).upper(), 0x00)

        # 发送数据并锁存输出
        self.latch.value(0)
        self._shift_out(code)
        self.latch.value(1)
        time.sleep_us(1)
        self.latch.value(0)

    def clear(self):
        """关闭所有段"""
        self.display(' ')

实验一：一位数码管的显示

程序如下：

'''
一位数码管上显“0~9”、“A~F”和小数点“.”
'''
 #
# 从我们存放第三方库文件的文件夹（libs）中导入SegmentDisplay
from libs.SegmentDisplay import SegmentDisplay
import time

# 初始化数码管 (DS=13, SHCP=14, STCP=15，共阴极)，配置对应的管脚
seg = SegmentDisplay(data_pin=23, clock_pin=5,latch_pin=18, common_cathode=True)

# 第一：显示数字 0-9
for num in range(10): 
    seg.display(num)
    time.sleep(0.5)

# 第二：显示字母 A-F
for char in ['A', 'B', 'C', 'D', 'E', 'F','.']:
    seg.display(char)
    time.sleep(1)

# 第三：显示带小数点的数字
seg.display('5.')  # 需在段码表中定义对应组合
time.sleep(5)

# 第四：关闭显示
seg.clear()

通过调用第三方库，我们就可以轻松实现常见的数字字母和小数点的显示

注意：

如果需要显示数字或字母同时显示“.”，就需要在库文件中，增加对应的组合

实验二：倒计时交通灯

多线程的介绍

MicroPython 通过 _thread 模块提供 轻量级多线程 支持，适用于并发执行简单任务。

_thread.start_new_thread(func, args) 启动新线程，args 为参数元组。

程序如下：

from libs.SegmentDisplay import SegmentDisplay
from machine import Pin
import time
import _thread

# 初始化数码管 (DS=13, SHCP=14, STCP=15，共阴极)，配置对应的管脚
display1 = SegmentDisplay(data_pin=23, clock_pin=5,latch_pin=18, common_cathode=True)

# 定义数码管线程
def display1_task():
    while True :
        for num in range(9,-1,-1):
            display1.display(num)
            time.sleep(1)
        for num in range(3,0,-1):
            display1.display(num)
            time.sleep(1)
        for num in range(9,-1,-1):
            display1.display(num)
            time.sleep(1)
        for num in range(3,0,-1):
            display1.display(num)
            time.sleep(1)


# 初始化交通灯引脚（根据实际接线修改）
red = Pin(27, Pin.OUT)    # 红灯
yellow = Pin(26, Pin.OUT) # 黄灯
green = Pin(25, Pin.OUT)  # 绿灯

# 初始化所有灯灭
red.value(0)
yellow.value(0)
green.value(0)

# 主程序
def traffic_light_task():

    while True:
        # 绿灯阶段
        green.value(1)   # 绿灯亮
        print("绿灯通行")
        time.sleep(10)
        green.value(0)  

        # 黄灯过渡，闪烁三次
        for led in range(3):
            yellow.value(1) # 黄灯亮
            time.sleep(0.5)
            print("黄灯等待")
            yellow.value(0)
            time.sleep(0.5)

        # 红灯阶段
        red.value(1)  # 红灯亮
        print("红灯禁止")
        time.sleep(10)
        red.value(0)

        # 黄灯过渡，闪烁三次（双向交通需要）
        for led in range(3):
            yellow.value(1) # 黄灯亮
            time.sleep(0.5)
            print("准备通行")
            yellow.value(0)
            time.sleep(0.5)



# 启动交通灯控制线程
_thread.start_new_thread(traffic_light_task, ())

# 启动数码管显示线程
_thread.start_new_thread(display1_task, ())
