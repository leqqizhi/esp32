  • 热敏传感器介绍

在公共场所，比如酒店，建筑物和其他地方都配备了火灾报警器，那么它如何感知火灾？ 众所周知，当火灾爆发时，会有特别强烈的红外线，该设备可以通过红外线探测火灾。​ 火焰传感器是机器人专门用来搜寻火源的传感器，当然火焰传感器也可以用来检测光线的亮度，只是本传感器对火焰特别灵敏。火焰传感器利用红外线对对火焰非常敏感的特点，使用特制的红外线接受管来检测火焰，然后把火焰的亮度转化为高低变化的电平信号，输入到中央处理器中，中央处理器根据信号的变化做出相应的程序处理。

  • 实物展示

  • 我们使用的这款火焰传感器是可以进行数字信号和模拟信号切换，D为数字信号，A为模拟信号（如上图）

  • 蓝色的方块上有个十字孔是用来调节灵敏度的。使用前，我们可以调节到蓝色指示灯刚好熄灭的状态。

实验一：检测信号数值

  • 信号检测

  • 数字信号检测

```python ''' 气体传感器信号检测（数字信号） ''' from machine import Pin,ADC import time

# 在 32 引脚创建 Pin 对象 gas = Pin(32, Pin.IN) while True: print(gas.value()) time.sleep(0.1)
```

测试结果：正常状态下显示数字：1；发现有毒显示数字：0

  1. 模拟信号检测

```python ''' 火焰传感器信号检测（模拟） ''' from machine import Pin,ADC import time

# 在 32 引脚创建 ADC 对象 gas = ADC(Pin(32, Pin.IN),atten=ADC.ATTN_11DB) while True: print(gas.read_u16()) time.sleep(0.1) ```

测试结果：没有火焰数值 <35000，出现火焰，数值 > 45000

实验二：煤气报警器

  • 数字信号程序如下
'''
煤气报警器（数字信号）
'''
from machine import Pin, ADC, PWM
import time

# 在 32 引脚创建 ADC 对象
gas = Pin(32, Pin.IN)
# 初始化PWM（频率1kHz）
pwm = PWM(Pin(2), freq=1000, duty=0)  # duty范围0-1023

# 第一个蜂鸣器报警程序
def beep_pwm(duration=1, volume=512):
    """通过PWM调节音量"""
    pwm.duty(volume)  # 音量设置（0静音，1023最大）
    time.sleep(duration)
    pwm.duty(0)

while True:
    print(gas.value())
    if gas.value() == 0:
        beep_pwm()

while True:
