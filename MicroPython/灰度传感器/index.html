  • 灰度传感器介绍**

灰度传感器是模拟传感器，与Arduino 结合使用，可以感知地面或桌面不同的颜色而产生相应的信号，可实现与颜色有相关的互动作品，也可以作为巡线小车的巡线传感器或者足球机器人的场地灰度识别。灰度传感器包括一个白色高亮发光二极管和一个光敏电阻，由于发光二极管照射到灰度不同的纸张上返回的光是不同的，光敏电阻接收到返回的光，根据光的强度不同，光敏电阻的阻值也不同，从而实现灰度值的测试。

实验一：检测信号数值

  • 信号检测

  • 模拟信号检测**

```python ''' 灰度传感器信号检测（模拟模式） ''' from machine import Pin, ADC import time

# 在 4 引脚创建 Pin 对象 Grayscale = ADC(Pin(2, Pin.IN),atten = ADC.ATTN_11DB) while True: print(Grayscale.read_u16()) time.sleep(0.1) ```

测试结果：颜色越深，值越小，不同颜色，值的范围不同

实验二：在四位数码管上显示灰度值

  • 程序如下
from machine import Pin, I2C, ADC
from libs.FourDigitDisplay import FourDigitDisplay  # 导入库
import time

# ======== 灰度传感器配置 ========
grayscale = ADC(Pin(2, Pin.IN),atten = ADC.ATTN_11DB)

# ======== TM1650数码管配置 ========
i2c = I2C(scl=Pin(22), sda=Pin(21), freq=100000)
display = FourDigitDisplay(i2c)


while True:
    value = grayscale.read_u16()

    # 在四位数码管上显示灰度值
    display.shownum(value)
    time.sleep(0.2)
