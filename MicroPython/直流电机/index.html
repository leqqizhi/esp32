上手实验

实验一：正反转测试

实验要求：正转2秒，反装2秒，然后停止

程序如下：

from machine import Pin, PWM
import time

pin16 = Pin(16,Pin.OUT)
pin17 = Pin(17,Pin.OUT)
pin2 = Pin(2,Pin.IN)

# 正转2秒
pin16.value(1)
pin17.value(0)
time.sleep(2)

# 反转2秒
pin16.value(0)
pin17.value(1)
time.sleep(2)

# 停止
pin16.value(0)
pin17.value(0)


实验二：使用按键控制电机转动

程序要求按键按下后电机开始转动，再次按下后电机电机停止转动

  • 程序如下：
from machine import Pin
import time

pin2 = Pin(2,Pin.IN)
pin16 = Pin(16,Pin.OUT)
pin17 = Pin(17,Pin.OUT)

while True:
    while pin2.value() == 1: # 按键防抖
        pass
    while pin2.value() == 0:
        pin16.value(1)
        pin17.value(0)
    while pin2.value() == 1: # 按键防抖
        pass
    while pin2.value() == 0:
        pin16.value(0)
        pin17.value(0)

ADC（数模转换器）简介：

ADC（Analog to Digital Converter）即数模转换器，其功能是将模拟信号转换为数字信号。ESP32单片机可使用ADC1、ADC2两组ADC，每组有多路ADC输入端，没路ADC可以看作一个单独的ADC，ADC1组的ADC输入端有32~39引脚，ADC2组的ADC输入端有2、4、12~15、25~27引脚，ESP32中ADC默认是将0~1.0v的电压转换成0~4095。

  • ADC的类和函数

  • 构建ADC对象

    machine.ADC(Pin(id))

    id 为ADC使用的引脚

    ADC组别 可用GPIO引脚 默认量程 分辨率 注意事项
    ADC1 32, 33, 34, 35,
    36, 37, 38, 39    		 0-1.0V → 0-4095 12位 WiFi开启时仍可用
    ADC2 0, 2, 4, 12, 13,
    14, 15, 25, 26, 27    		 0-1.0V → 0-4095 12位 启用WiFi后不可用
    2. ADC的衰减

ADC默认只能输入转换0~1v的电压，要测量其他范围的电压可以对输入电压进行衰减。ADC衰减使用ADC类中的atten()函数：

ADC.atten(atten)

序号 参数值 衰减量 测量范围
1 ADC.ATTEN_0DB 0 衰减 0~1.0v
2 ADC.ATTEN_2_5DB 2.5dB 衰减 0~1.34v
3 ADC.ATTEN_6DB 6dB 衰减 0~2v
4 ADC.ATTEN_11DB 11dB 衰减 0~3.6v
  1. ADC的分辨率

ADC分辨率指ADC值的位数，位数越多，ADC值反应的模拟电压值越精确。ADC分辨率使用ADC类中的width()函数，语法格式：

ADC.width(bits)

bits用于指定分辨率。

序号 参数值 位 值范围
1 ADC.WIDTH_9BIT 9 0~511
2 ADC.WIDTH_10BIT 10 0~1023
3 ADC.WIDTH_11BIT 11 0~2047
4 ADC.WIDTH_12BIT 12 0~4095
  1. 读取ADC值

读取ADC值，使用ADC类中的read()函数：

ADC.read()

该函数返回指定分辨率的模拟电压转换成的ADC值，默认分辨率为12位，ADC的值范围为0~4095

序号 返回值类型 ADC值范围
1 ADC.read() 0~4095（整数）
2 ADC.read_uv() 毫伏
3 ADC.read_u16() 0~65536（整数）
- 代码示例: 
# 从machine模块中导入Pin类和ADC类
from machine import Pin,ADC

# 示例化一个ADC对象
adc1 = ADC(Pin(35))
adc1.read() # 按照默认配置读取电压转换成ADC值（0~4095表示0~1.0V电压）

# 将ADC衰减量设置为11DB，转换成0~3.6v范围的模拟电压
adc1.atten(ADC.ATTN_11DB)
adc1.read() # 按照的新衰减量读取转换后的ADC值

实验程序代码：

from machine import Pin,ADC,PWM
import time

# 初始化引脚
encoder = ADC(Pin(32))  # 旋钮接32号引脚
# 将ADC衰减量设置为11DB，转换成0~3.6v范围的模拟电压
encoder.atten(ADC.ATTN_11DB)
motor_a = PWM(Pin(17), freq=1000)      # 电机引脚A
motor_b = PWM(Pin(16), freq=1000)      # 电机引脚B

value = 0 # 初始值为0
speed = 0 # 初始速度为0

# 定义一个映射函数        
def math_map(x, in_min, in_max, out_min, out_max):
    """
    将 x 从 [in_min, in_max] 线性映射到 [out_min, out_max]
    """
    return (x - in_min) * (out_max - out_min) / (in_max - in_min) + out_min

while True:
    value = encoder.read()  # 值的范围为0~4095
    speed = int(math_map(value,0,4095,0,1023))
    print(speed)
    motor_a.duty(speed)
    motor_b.duty(0)
