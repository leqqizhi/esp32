  • 超声波传感器介绍

HS-SR04超声波模块是在传统的HC-SR04超声波模块基础上进行优化设计。功能尺寸大小完全兼容传统的HC-SR04模块，在细节上比传统的HC-SR04模块更优化了许多。本模块性能稳定，测度距离精确，模块高精度，盲区小。产品应用领域：机器人避障、物体测距、液位检测、公共安防、停车场等等····

  • 第三方库

我们在libs文件夹中新建一个HCSR04.py文件，把网上的地方放库保存进去就可以使用了：

```python from machine import Pin, time_pulse_us import time

class HCSR04: """ HCSR04超声波测距模块驱动类 适用于MicroPython ESP32环境 引脚说明： trig_pin: 触发信号引脚（输出模式） echo_pin: 回声接收引脚（输入模式） 注意：ESP32引脚工作电压为3.3V，建议在Echo引脚添加分压电路避免损坏 """

  def __init__(self, trig_pin, echo_pin):
      """
      初始化超声波模块
      :param trig_pin: 触发引脚编号（整数）
      :param echo_pin: 回声引脚编号（整数）
      """
      # 初始化Trig引脚为输出模式
      self.trig = Pin(trig_pin, Pin.OUT)
      # 初始化Echo引脚为输入模式
      self.echo = Pin(echo_pin, Pin.IN)
      # 设置默认声速（25摄氏度干燥空气中的速度，单位：米/秒）
      self.sound_speed = 340
      # 设置超时时间（对应最大测量距离4米）
      self.timeout = 4 * 2 / self.sound_speed * 1000000  # 计算超时时间（微秒）

  def distance_cm(self, samples=3):
      """
      获取测量距离（厘米）
      :param samples: 采样次数（取多次测量的平均值）
      :return: 距离（厘米），-1表示测量失败
      """
      distances = []
      for _ in range(samples):
          distance = self._single_measurement()
          if distance == -1:
              continue
          distances.append(distance)
          time.sleep_ms(50)  # 每次测量间隔50ms

      if not distances:
          return -1  # 所有测量均失败
      return sum(distances) / len(distances)  # 返回平均值

  def _single_measurement(self):
      """
      执行单次距离测量
      :return: 单次测量距离（厘米），-1表示失败
      """
      # 确保Trig引脚先置低
      self.trig.value(0)
      time.sleep_us(2)

      try:
          # 发送10us高电平脉冲触发测量
          self.trig.value(1)
          time.sleep_us(10)
          self.trig.value(0)

          # 使用脉冲宽度测量函数获取高电平持续时间
          # 参数说明：引脚对象，触发状态，超时时间（微秒）
          pulse_time = time_pulse_us(self.echo, 1, int(self.timeout))

          # 计算距离（单位：厘米）
          # 公式说明：距离 = (时间 * 声速) / 2 （来回距离）
          # 时间单位转换：1秒=1e6微秒 → 厘米转换系数：0.0343/2 → 0.01715 → 1/58
          if pulse_time < 0:
              return -1  # 超时返回-1

          distance = (pulse_time * 0.0343) / 2  # 更精确的厘米计算方式
          # distance = pulse_time / 58  # 简化的近似计算方式

          # 过滤异常值（根据模块规格，有效测量范围2cm-400cm）
          if 2 <= distance <= 400:
              return distance
          return -1

      except Exception as e:
          print("测量错误:", e)
          return -1

# 使用示例 if name == "main": # 初始化传感器（Trig引脚=12, Echo引脚=14） sensor = HCSR04(trig_pin=12, echo_pin=14)

  while True:
      dist = sensor.distance_cm()
      if dist != -1:
          print("距离：{:.1f} cm".format(dist))
      else:
          print("测量失败")
      time.sleep(1)

```

实验1：超声波测距

程序如下

'''
实验一：在四位数码管上显示超声波检测距离
'''
from machine import I2C, Pin    
from libs.HCSR04 import HCSR04  # 导入超声波的库
from libs.FourDigitDisplay import FourDigitDisplay  # 导入四位数码管的库
import time                 


# 初始化I2C
i2c = I2C(scl=Pin(22), sda=Pin(21), freq=100000)
display = FourDigitDisplay(i2c)

# 初始化传感器（Trig引脚=32, Echo引脚=33）
sensor = HCSR04(trig_pin=32, echo_pin=33)

while True:
    dist = int(sensor.distance_cm()*100) # 将显示的数字转换为4为整数
    display.shownum(dist)
    display.showDP(bit=1, show=True) # 在第二位上显示小数点
    time.sleep(0.02)
    print(dist)
