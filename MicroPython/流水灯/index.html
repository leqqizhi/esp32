实验一：经典流水灯

  • 通过程序控制多颗LED依次点亮，然后依次熄灭形成流水般动态光效

  • 目标效果

逐个点亮： ●○○○ → ●●○○ → ●●●○ → ●●●●

逐个熄灭： ○●●● → ○○●● → ○○○● → ○○○○

  • 程序如下：
# 定义 led_pin_list 列表，保存 LED 管脚配置对象
led_pin_list = []

# 循环给 led_pin_list 列表添加对象
for i in pin_index_list:
    led_pin_list.append(Pin(i, Pin.OUT))

# LED全熄灭
for led_pin in led_pin_list:
    led_pin.value(0)

while True:

    # LED逐个点亮
    for led_pin in led_pin_list:
        led_pin.value(1)
        time.sleep(1)

    # LED逐个熄灭
    for led_pin in led_pin_list:
        led_pin.value(0)
        time.sleep(1)


实验二：not逻辑运算优化经典流水灯

在MicroPython中，not 是布尔逻辑运算符而非函数，用于对布尔值进行取反操作。

  • not语法讲解：
表达式 结果 说明
not True False 对True取反
not False True 对False取反
not 0 True 0在布尔上下文中等价于False
not 1 False 非零整数在布尔上下文中等价于True
  • 优化后程序如下
'''
第二种：常规流水灯（程序优化）
该程序作用是让 LED 依次点亮后依次熄灭
'''

import time
from machine import Pin

# 定义 LED 控制引脚
pin_index_list = [2, 4, 12, 13]

# 定义 led_pin_list 列表，保存 LED 管脚配置对象
led_pin_list = []

# 循环给 led_pin_list 列表添加对象
for i in pin_index_list:
    led_pin_list.append(Pin(i, Pin.OUT))

# LED全熄灭
for led_pin in led_pin_list:
    led_pin.value(0)

实验三：反复流水灯

  • 通过程序控制LED灯逐个点亮，然后从尾部开始这个熄灭

  • 目标效果

逐个点亮： ●○○○ → ●●○○ → ●●●○ → ●●●●

逐个熄灭： ●●●○ → ●●○○ → ●○○○ → ○○○○

  • 程序如下
'''
反复流水灯
该程序作用是让 LED 往复亮灭
'''

import time
from machine import Pin

# 定义 LED 控制引脚
pin_index_list = [2, 4, 12, 13]

# 定义 led_pin_list 列表，保存 LED 管脚配置对象
led_pin_list = []

# 循环给 led_pin_list 列表添加对象
for i in pin_index_list:
    led_pin_list.append(Pin(i, Pin.OUT))

# LED全熄灭
for led_pin in led_pin_list:
    led_pin.value(0)

while True:

    # LED逐个点亮
    for led_pin in led_pin_list:
        led_pin.value(1)
        time.sleep(1)

    # LED逐个熄灭
    for led_pin in reversed(led_pin_list): # reversed的作用是使得列表顺序翻转
        led_pin.value(0)
        time.sleep(1)

实验四：移动流水灯

  • 通过程序控制LED灯逐个点亮，点亮后熄灭，然后下一盏灯立即点亮

  • 目标效果

逐个点亮： ●○○○ → ○●○○ → ○○●○ → ○○○●

看上去亮灯在移动： ●○○○ → ○●○○ → ○○●○ → ○○○●

  • 程序如下
'''
LED 的移动点亮（从头到尾不停循环）
该程序作用是让 LED 往复亮灭
'''

import time
from machine import Pin

# 定义 LED 控制引脚
pin_index_list = [2, 4, 12, 13]

# 定义 led_pin_list 列表，保存 LED 管脚配置对象
led_pin_list = []

# 循环给 led_pin_list 列表添加对象
for i in pin_index_list:
    led_pin_list.append(Pin(i, Pin.OUT))

# 获取 LED_Pin_list 的长度
num = len(led_pin_list)

# LED全熄灭
for led_pin in led_pin_list:
    led_pin.value(0)

while True:

    for i in range(num):
        # LED逐个点亮
        led_pin_list[i].value(1)
        # 如果这颗 LED 是第一个，则需要改变最后一颗 LED 的状态
        if i == 0:
            led_pin_list[num - 1].value(0)
        # 如果这颗 LED 不是第一个，则需要改变它之前一颗 LED 的状态
        else:
            led_pin_list[i - 1].value(0)

        # 延时 1 秒
        time.sleep(1)

实验五：往复移动流水

  • 通过程序控制LED灯逐个点亮，点亮后熄灭，然后下一盏灯立即点亮，到最后一盏灯后，再从后往前逐个点亮。点亮后熄灭，然后立即点亮前面一盏灯

  • 目标效果

亮灯从前往后移动： ●○○○ → ○●○○ → ○○●○ → ○○○●

亮灯从后往前移动： ○○○● → ○○●○ → ○●○○ → ●○○○

  • 程序如下
'''
跑马灯：LED 的移动点亮（从头到尾不停循环）
该程序作用是让 LED 往复亮灭
'''

import time
from machine import Pin

# 配置LED引脚（根据实际硬件修改）
led_pins = [Pin(2, Pin.OUT),
            Pin(4, Pin.OUT),
            Pin(12, Pin.OUT),
            Pin(13, Pin.OUT)]

# 初始化所有LED灯熄灭
for led in led_pins:
    led.value(0)

# 永真循环
while True:
    # 正向流动，从头到尾逐个点亮并立即熄灭
    for i in range(len(led_pins)):
        led_pins[i].value(1) # 点亮当前的LED灯
        time.sleep(0.5)      # 保持灯亮0.5秒
        led_pins[i].value(0) # 熄灭当前LED灯

    # 方向流动，从尾到头逐个点亮并立即熄灭
    for i in reversed(range(len(led_pins))):
        led_pins[i].value(1) # 点亮当前的LED灯
        time.sleep(0.5)      # 保持灯亮0.5秒
        led_pins[i].value(0) # 熄灭当前LED灯

实验六：对实验五进行面向对象式编程改造

什么是面向对象编程：

是一种以“对象”为核心的编程范式，将现实事物抽象为具有属性和行为的独立单元。其核心特性包括：

  1. 封装

将数据和方法封装在对象内，通过接口暴露功能（如汽车对象隐藏发动机细节，仅提供启动()方法）。

  1. 继承

支持类层级扩展（如动物→猫科→猫），实现代码复用和逻辑分层。

  1. 多态

同一方法在不同对象中表现不同行为（如动物.叫()，狗吠猫喵）。

  1. 抽象

定义接口规范（如图形类声明计算面积()，由子类具体实现）。

第一步：封装数据和逻辑

在项目文件夹中新建一个文件夹common（名字自己取，要求是英文）用来放置我们自己写的类文件，然后在文件夹中，新建一个Python文件，我们示例为：pm_led.py

  • 程序如下：
import time
from machine import Pin
class LEDController:
    '''
    初始化LED灯控制器
    :param pins: led引脚编号 例如 [1,2,3,4]
    :param delay: 点亮时间间隔（毫秒）
    '''
    def __init__(self,pins,delay=150):
        self.delay = delay
        self.leds = []
        for i in pins:
            a= Pin(i, Pin.OUT)
            self.leds.append(a)

    def all_off(self):
        # 熄灭所有的灯
        for led in self.leds:
            led.value(0)
    def flow_forward(self):
        # 正向流动效果
        for led in self.leds:
            led.value(1)  # 点亮当前LED灯
            time.sleep_ms(self.delay)
            led.value(0)  # 熄灭当前LED灯

    def flow_backward(self):
        # 方向流动效果
        for led in self.leds[::-1]:
            led.value(1)  # 点亮当前LED灯
            time.sleep_ms(self.delay)
            led.value(0)  # 熄灭当前LED灯

    def run(self):
        # 运行主循环
        while True:
            self.flow_forward() # 正向扫描
            self.flow_backward()# 反向扫描



# 使用__name__构造函数来对类函数进行调试，而其他程序调用本类中的方法，此构造函数不会被执行
if __name__ == "__main__":
    controller = LEDController(pins=[2,4,12,13],delay=500)
    controller.run()

对于这个程序暂时大家可以不要求会自己编写，但是要确保可以看懂，能理解面向对象编程的思想，这里我们将引脚对象封装为一个pins的列表、延时时间封装为delay属性，并给与150毫秒的默认值，后面我们调用的时候根据实际情况直接传入。我们将流水灯的程序封装整合成一个run方法。

  • 再重新建立一个文件调用刚才的类
'''
将过程式的代码修改为面向对象的形式
跑马灯：LED 的移动点亮（从头到尾不停循环）
该程序作用是让 LED 往复亮灭
'''
# 首先需要导入类文件
from common.pm_led import LEDController

# 实例化对象，并传入对应的参数，一般使用关键字传参的方法
controller = LEDController(pins=[2,4,12,13],delay=500)
# 调用类方法
controller.run()
