  • 声音传感器介绍

声音传感器模块的作用相当于一个话筒(麦克风)。它用来接收声波，显示声音的振动图象，但不能对噪声的强度进行测量。该传感器内置一个对声音敏感的电容式驻极体话筒。声波使话筒内的驻极体薄膜振动，导致电容的变化，而产生与之对应变化的微小电压。这一电压随后被转化成0-5V的电压，经过A/D转换被数据采集器接受，并传送给主控芯片。

  • 实物展示

  1. 我们使用的这款声音传感器是可以进行数字信号和模拟信号切换，D为数字信号，A为模拟信号（如上图）

  2. 蓝色的方块上有个十字孔是用来调节灵敏度的。使用前，我们可以调节到蓝色指示灯刚好熄灭的状态。

实验一：检测信号数值

  • 信号检测

  • 数字信号检测

```python ''' 声音传感器信号检测（数字信号） ''' from machine import Pin,ADC import time

# 在 32 引脚创建 Pin 对象 vol = Pin(32, Pin.IN) while True: print(vol.value()) time.sleep(0.1) ```

测试结果：安静状态下，值显示为“1”，当在有突发声音的时候，返回值为"0"

  1. 模拟信号检测

```python ''' 声音传感器信号检测（模拟） ''' from machine import Pin,ADC import time

# 在 32 引脚创建 ADC 对象 vol = Pin(32, Pin.IN) vol = ADC(Pin(32, Pin.IN),atten=ADC.ATTN_11DB) while True: print(vol.read_u16()) time.sleep(0.1) ```

测试结果：1. 安静时候的数字范围大概为：10000~12000

2. 有突发声响的时候数值会低于10000

实验二：声控灯

  • 声控灯（模拟信号）
'''
实验1：声控灯（模拟信号）
'''
from machine import Pin,ADC
import time

# 在 32 引脚创建 ADC 对象
vol = Pin(32, Pin.IN)
led = Pin(2,Pin.OUT)

vol = ADC(Pin(32, Pin.IN),atten=ADC.ATTN_11DB)

while True:
    if vol.read_u16() < 10000:
        led.value(1)
        time.sleep(5)
        led.value(0)
  • 声控灯（数字信号）
'''
实验1：声控灯（数字信号信号）
'''
from machine import Pin,ADC
import time

# 在 32 引脚创建 ADC 对象
vol = Pin(32, Pin.IN)
led = Pin(2,Pin.OUT)

vol = Pin(32, Pin.IN)
while True:
    if vol.value() == 0:
        led.value(1)
        time.sleep(5)
        led.value(0)
