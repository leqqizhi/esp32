PWM是什么

  • 1. 核心概念

  • 脉冲宽度调制：通过调节信号的占空比（高电平时间占比）模拟不同电压效果

  • 关键参数：

    • 频率：每秒脉冲周期数（Hz）

    • 占空比：高电平时间占周期的百分比（0-100%）

  • 公式

    等效电压 = 占空比 × 最大电压

  • 2. 波形图

  • 3. 关键参数
参数 说明 典型应用场景
频率 每秒脉冲周期数 (Hz) 电机控制(1k-20kHz)
占空比 高电平时间占比 (0-100%) LED调光、电源稳压
分辨率 占空比调节精度 (如8位=256级) 精密控制场景

PWM 在 machine 的 PWM 模块中，我们也是只需要了解其构造对象函数和使用方法：

构造函数 machine.PWM(dest, freq, duty, duty_u16, duty_ns)，使用以下参数构造并返回一个新的 PWM 对象：

  • dest 是输出 PWM 的实体，通常是 machine.Pin 对象；
  • freq 应该是一个整数，用于设置 PWM 周期的频率（以 Hz 为单位）；
  • duty 占空比，范围是 0 - 1023；
  • duty_u16 占空比，范围是 0 - 65535，2 的 16 次方；
  • duty_ns 以纳秒为单位设置脉冲宽度，范围是 0 - 50000。

MicroPython PWM实现

# 和Pin一样属于machine模块，使用前需要导入
from machine import Pin, PWM

# 创建PWM对象
led = PWM(Pin(12), freq=20000, duty=512)


led.freq()  # 获取当前频率

led.freq(1000) # 设置频率

led.duty()  # 获取当前占空比

led.duty(200)  # 设置占空比

led.duty_u16(12345)  # 使用 duty_u16 方法

led.duty_ns()  # 使用 duty_ns 方法

led.deinit()   # 关闭引脚的 PWM

实验一：呼吸灯

呼吸灯是一种通过PWM（脉冲宽度调制）技术实现LED亮度渐变循环的灯光效果，模拟人类呼吸的明暗节奏

  • 程序如下：
'''
交通灯3色灯轮流点亮
'''
import time
from machine import Pin, PWM


# 创建 LED 控制对象
led = PWM(Pin(2), freq=1000)


while True:
    # 渐亮
    for i in range(0, 1024):
        led.duty(i)
        time.sleep_ms(1)

    # 渐暗
    for i in range(1023, 0, -1):
        led.duty(i)
        time.sleep_ms(1)


实验二：面向对象实现呼吸灯

在项目文件夹中新建一个文件夹common（名字自己取，要求是英文）用来放置我们自己写的类文件，然后在文件夹中，新建一个Python文件，我们示例为：BreathLED

  • 程序如下：
import time
from machine import Pin, PWM

class BreathLED:

    def __init__(self, pin, freq=1000, max_duty=1023, speed=3):
        """
        呼吸灯（从明到暗再从暗到明）
        :param pin: 引脚编号
        :param freq: 频率值（默认为1000）
        :param max_duty: 最大占空比范围（0~1023）
        :param speed: 控制呼吸灯节奏
        """
        self.pwm = PWM(Pin(pin))
        self.pwm.freq(freq)       # 设置PWM频率
        self.max_duty = max_duty  # 最大亮度值（0-1023）
        self.steps = 100          # 呼吸平滑度
        self.speed = speed        # 呼吸速度（秒/步）
    def linear(self):
        while True:
            # 渐亮
            for i in range(0, 1024):
                self.pwm.duty(i)
                time.sleep_ms(self.speed)

            # 渐暗
            for i in range(1023, 0, -1):
                self.pwm.duty(i)
                time.sleep_ms(self.speed)


# 使用__name__构造函数来对类函数进行调试，而其他程序调用本类中的方法，此构造函数不会被执行
if __name__ == "__main__":
    # 示例化对象
    led = BreathLED(pin=2,speed=5)
    led.linear()
  • 再重新建立一个文件调用刚才的类
'''
PWM呼吸灯
pin：对应引脚编号
speed：对应的呼吸速度，必须为整数型，单位（毫秒）
'''
from common.BreathLED import BreathLED


led = BreathLED(pin=2,speed=1)
led.linear()
