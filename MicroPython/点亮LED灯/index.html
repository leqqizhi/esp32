涉及语法讲解

  • Pin函数讲解

MicroPython 中可使用 machine 模块中的 Pin 模块对 GPIO 输出控制。

构造函数Pin(id, mode=-1, pull=-1, value, drive, alt)相关参数讲解：

参数 类型/选项 说明
id int 引脚编号（硬件相关）：
- ESP32：使用 GPIO 编号（如 2 对应 IO2）
- Pico：物理引脚号（如 0 对应 GP0）
mode Pin.IN
Pin.OUT
Pin.OPEN_DRAIN
Pin.ALT		 模式：
- IN：输入模式（默认）
- OUT：推挽输出
- OPEN_DRAIN：开漏输出（适合 I2C）
- ALT：复用功能（如 UART）
pull Pin.PULL_UP
Pin.PULL_DOWN
None		 上下拉电阻（仅输入模式有效）：
- 上拉：默认高电平
- 下拉：默认低电平
- None：禁用（需外接电阻）
value 0/1 初始电平（仅输出模式有效）：
- 0：初始低电平
- 1：初始高电平
drive 0（弱）~3（强） 驱动能力（部分芯片支持）：
设置输出电流强度（如 Pico 的 drive=Pin.DRIVE_3 对应 12mA）
alt int（功能编号） 复用功能选择：
指定引脚的特殊功能（如 ESP32 的 alt=5 用于 SPI CLK）

其他参数在初级阶段涉及相对较少，更多内容可以参考官网文档

  • MicroPython GPIO 控制示例:
from machine import Pin

p0 = Pin(0, Pin.OUT)  # 初始化 GPIO0 为输出模式

from machine import Pin

p0 = Pin(0, Pin.OUT)  # 初始化 GPIO0 为输出模式

# 电平控制操作
p0.value(0)  # 输出低电平（0V） 
p0.value(1)  # 输出高电平（3.3V）

# 等效操作（推荐写法）
p0.on()      # 等同于 value(1)
p0.off()     # 等同于 value(0)

# 初始化 GPIO2 为输入模式（启用内部上拉电阻）
p2 = Pin(2, Pin.IN, Pin.PULL_UP)  

# 读取引脚状态
print("GPIO2 当前电平:", p2.value())  # 返回 0（低电平）/1（高电平）

  • 代码解析

  • 引脚输出控制

    代码 说明
    Pin(0, Pin.OUT) 将 GPIO0 配置为推挽输出模式，可主动输出高/低电平
    .value(0) 输出低电平（0V），常用于关闭 LED 或继电器
    .value(1) 输出高电平（3.3V），常用于点亮 LED 或激活设备

  • 输入信号检测

    代码 说明
    Pin(2, Pin.IN, Pin.PULL_UP) 将 GPIO2 配置为输入模式并启用内部上拉电阻，悬空时默认高电平
    p2.value() 读取当前电平状态：
      -  1：高电平（按钮释放）
      -  0：低电平（按钮按下）

通过上面的文档我们知道，想要让一个引脚输出高电平，只需要找到对应的 GPIO 然后通过 on() 或者 value(1) 操作就可以，同理如果想要输出低电平让 LED 灯灭，只需要调用 off() 或者 value(0) 就行。

上手实验

实验一：点亮一颗 LED灯

我们想要点亮这颗 LED 灯，只需要先构建引脚对象，然后给这个引脚赋值一个高电平即可。

'''
点亮1颗LED灯
'''

from machine import Pin

# 构建 p2 引脚对象，GPIO2输出
p2 = Pin(2, Pin.OUT)

# 使 P2 输出高电平，点亮LED
p2.value(1)

通过 Thonny 编写上述代码，然后运行，此时会看到电路中的 LED 灯被点亮了。


实验二：闪烁LED灯

我们已经成功点亮一颗 LED ，接下来，可以尝试让这颗 LED 不停的闪烁。

实现 LED 闪烁的原理很简单，就是在循环语句中使用延时模块。先设置高电平，延时 X 秒，再设置低电平，延时 X 秒，之后就不断循环该语句即可。

我们知道循环语句与两种： for 和 while 循环语句，如果我们想要让灯泡一直闪烁，则需要设置无限循环，因此使用 while 更合适。

# 如果 while 的条件为 True，一直为真，就可以实现无限循环了。
while True:
    pass
  • 概述

MicroPython 的 time 模块提供基础时间管理功能，专为嵌入式设备优化，支持毫秒/微秒级延时、系统时间戳获取及时区转换。适用于定时任务、传感器数据采集同步等场景。

  • 核心方法
方法 功能说明
time.sleep(seconds) 阻塞延时（秒级），支持浮点数（如 sleep(0.5) 延时 500ms）
time.sleep_ms(ms) 毫秒级延时（整数参数）
time.sleep_us(us) 微秒级高精度延时（ESP32 等支持）
time.time() 返回系统时间戳（秒级，从 2000/1/1 开始计时，依赖 RTC 模块或网络同步）
time.localtime() 返回本地时间的元组 (年, 月, 日, 时, 分, 秒, 周, 年日)
time.ticks_ms() 获取系统运行毫秒计数器（自动溢出循环，用于精准相对计时）
time.ticks_diff(end, start) 计算两个 ticks 的时间差（正确处理计数器溢出）

  • 应用场景:

  • 定时触发：结合 Timer 硬件外设实现周期性任务（如每 10s 上报传感器数据）

  • 性能调试：使用 ticks_ms() 测量代码执行时间

# 导入 time 模块
import time

# 延时 0.5 秒
time.sleep(0.5)

# 延时 100 毫秒
time.sleep_ms(100)

# 延时 100 微秒
time.sleep_us(100)

# 获取毫秒计时器当前值
time_1 = time.ticks_ms()

这里的 time 模块是 MicroPython 中的 time 模块，与 Python 中的不同，更多使用方法可以参考官方文档 MicroPython Time 模块

所以，我们的程序可以这么写：

# 导入time模块
import time

# 导入 Pin 模块
from machine import Pin

# 构建 P2 对象，GPIO2输出
p2 = Pin(2, Pin.OUT)  

# 永真循环
while True: 

    # 使 P2 输出高电平，点亮 LED
    p2.on()

    # 延时 0.5 秒
    time.sleep(0.5)

    # 使 P2 输出低电平，熄灭 LED
    p2.off()

    time.sleep(0.5)

运行程序，LED 就闪烁了。

实验三：SOS求救信号灯

  • SOS求教信号程序要求：

  • LED灯先快速亮灭3次，然后缓慢亮灭3次，再快速亮灭3次，并且一直重复上述现象。

  • 案例展示 可联系我索要拼搭图纸。

第一种实现方式：
'''
第一种常规方案
求救信号灯
要求3次短闪（间隔500毫秒），再3次长闪烁（间隔1000毫秒），然后再次快闪
以上为一次SOS信号
间隔5秒之后，重复发送SOS求救信号
'''
import machine  # 导入machine模块

import time # 导入time模块

# 构建 Pin2 对象，GPIO2输出
pin2 = machine.Pin(2, machine.Pin.OUT)

while True:

    # 实现3次短间隔闪烁
    for i in range(0, 3, 1):
        pin2.value(1)
        time.sleep_ms(500)
        pin2.value(0)
        time.sleep_ms(500)

    # 实现3个长间隔闪烁
    for i in range(0, 3, 1):
        pin2.value(1)
        time.sleep_ms(1000)
        pin2.value(0)
        time.sleep_ms(1000)

    # 实现3次短间隔闪烁
    for i in range(0, 3, 1):
        pin2.value(1)
        time.sleep_ms(500)
        pin2.value(0)
        time.sleep_ms(500)

    # 间隔5秒重复发送SOS求救信号
    time.sleep(5)
第二种实现方式（优化程序）：
'''
第二种：优化方案
求救信号灯
要求3次短闪（间隔500毫秒），再3次长闪烁（间隔1000毫秒），然后再次快闪
以上为一次SOS信号
间隔5秒之后，重复发送SOS求救信号
'''
import machine  # 导入machine模块

import time # 导入time模块

# 构建 Pin2 对象，GPIO2输出
pin2 = machine.Pin(2, machine.Pin.OUT)

while True:

    # 实现3次短间隔闪烁
    for i in range(0, 3):
        pin2.value(not(pin2.value()))
        time.sleep_ms(500)


    # 实现3个长间隔闪烁
    for i in range(0, 3, 1):
        pin2.value(not(pin2.value()))
        time.sleep_ms(1000)


    # 实现3次短间隔闪烁
    for i in range(0, 3, 1):
        pin2.value(not(pin2.value()))
        time.sleep_ms(500)


    # 间隔5秒重复发送SOS求救信号
    pin2.value(0)
    time.sleep(5)
