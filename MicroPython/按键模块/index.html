按键模块介绍

按键模块是电子系统中用于实现人机交互的基础输入组件，用于检测用户按压动作并将机械信号转换为电信号。

实验一：按键控制LED灯的亮灭

  • 代码如下：
'''
按键按下灯亮，松开灯灭
按键模块GPIO 2号引脚
LED模块 GPIO 4号引脚
'''

from machine import Pin
import time

led = Pin(4, Pin.OUT)   # LED灯接 GPIO 4
button = Pin(2, Pin.IN)   # 按键接 GPIO 2


while True:
    # 测试按键的返回值
    print(button.value())  
    time.sleep(0.1)   # 默认状态为：1 按下值为：0

    if not button.value(): # 按键发下时
        time.sleep_ms(10)  # 按键防抖
        led.value(1)   # 按键按下后，灯亮

    else :
        led.value(0) # 按键松开后，灯灭

实验二：每按一次按键LED灯状态改变一次

  • 代码如下：
'''
按键控制LED灯，每按一次LED灯改变
次按键模块GPIO 2号引脚
LED模块 GPIO 4号引脚IO4
'''

from machine import Pin
import time

led = Pin(4, Pin.OUT)   # LED灯接 GPIO 4
button = Pin(2, Pin.IN)   # 按键接 GPIO 2

num = True # 控制灯亮灭的逻辑
while True :
    if not button.value(): # 按键发下时
        time.sleep_ms(10)  # 按键防抖
        while button.value() == 0:   # 按键按下未释放
            pass
        num = not num # 检测到按键动作后，灯亮模式发生改变
        if num:
            led.value(1)   # 按键按下后，灯亮

        else :
            led.value(0) # 按键松开后，灯灭

实验三：通过外部中断实现按键控制LED灯

  • 什么是中断

当单片机正在执行某个程序时，如果突然出现意外情况需要停止当前的工作去执行其他指定的程序（中断程序或中断函数），执行完成后，再回来继续执行原来的程序，这就叫中断。

  • 中断函数

ESP32使用了Pin类中Pin.irq()函数来实现外部中断程序：

pin.irq(handler=None, trigger=(Pin.IRQ_FALLING | Pin.IRQ_RISING))

参数名 说明 可选值/示例 注意事项
handler 中断处理函数 def isr(pin): ... 应保持简短，避免阻塞操作
trigger 触发条件 Pin.IRQ_FALLING（下降沿）
Pin.IRQ_RISING（上升沿）
组合使用 \| 运算符		 电平触发模式（LOW/HIGH）可能导致持续中断

触发类型对照表

触发类型常量 触发条件描述 适用场景
Pin.IRQ_FALLING 下降沿触发（高电平→低电平） 常规按键检测
Pin.IRQ_RISING 上升沿触发（低电平→高电平） 释放检测
Pin.IRQ_LOW_LEVEL 低电平持续触发 长按检测
Pin.IRQ_HIGH_LEVEL 高电平持续触发 特殊传感器信号
IRQ_FALLING丨IRQ_RISING 双边沿触发 需要同时检测按下/释放的场景

如何判断是上拉还是下拉

  • Pin.PULL_UP选择上拉：当需要引脚悬空时保持‌高电平（如按键未按下状态）

  • Pin.PULL_DOWN选择下拉：当需要引脚悬空时保持‌低电平（如传感器未触发状态）

我们通过一下程序查看按键的电位变化来判断:

import time

button = Pin(2, Pin.IN) # 开关链接2号引脚
while True :
    print(button.value())
    time.sleep(0.5)

实验三，松开按钮是改变LED灯状态

程序如下：

'''
松开按钮是触发中断程序
'''

import time
from machine import Pin

# 我的按钮默认是高电频，所以使用Pin.PULL_UP
button = Pin(2, Pin.IN, Pin.PULL_UP) 

led = Pin(4, Pin.OUT)
led.value(0) # 默认我灯灭状态


#定义 button 的外部中断函数
def button_irq(button):
    time.sleep_ms(80)
    if button.value() == 1: # 注意此时松开时候值为1
        led.value(not led.value())

# 松开按键是让电频从低至高，所以是上升沿
button.irq(button_irq, Pin.IRQ_RISING)

'''
两个按键控制LED灯效果
'''
from machine import Pin
import time

k1 = Pin(2, Pin.IN, Pin.PULL_UP)
k2 = Pin(4, Pin.IN, Pin.PULL_UP)

led1 = Pin(12, Pin.OUT)
led2 = Pin(13, Pin.OUT)
led3 = Pin(14, Pin.OUT)

led1_state, led2_state, led3_state = 0, 0, 0

def k1_irq(K1):
    global led1_state
    time.sleep_ms(10)
    if k1.value() == 0:
        led1_state = not led1_state
        led1.value(led1_state)

def k2_irq(K2):
    global led2_state
    time.sleep_ms(10)
    if k2.value() == 0:
        led2_state = not led2_state
        led2.value(led2_state)

if __name__ == "__main__":
    led1.value(0)
    led2.value(0)
    k1.irq(k1_irq, Pin.IRQ_FALLING)
    k2.irq(k2_irq, Pin.IRQ_FALLING)

    while True :
        led3.value(0)
        time.sleep(0.5)
        led3.value(1)
        time.sleep(0.5)

'''
4P按键分别控制4盏led灯
'''
from machine import Pin
import time

# 初始化按键（输入模式，无内部上拉，需外接上拉电阻）
buttons = [
    Pin(34, Pin.IN),  # 按键1 -> GPIO34
    Pin(35, Pin.IN),  # 按键2 -> GPIO35
    Pin(36, Pin.IN),  # 按键3 -> GPIO36
    Pin(39, Pin.IN)   # 按键4 -> GPIO39
]

# 初始化LED（输出模式，默认高电平熄灭）
leds = [
    Pin(2, Pin.OUT),   # LED1 -> GPIO2
    Pin(4, Pin.OUT),   # LED2 -> GPIO4
    Pin(12, Pin.OUT),  # LED3 -> GPIO12
    Pin(13, Pin.OUT)   # LED4 -> GPIO13
]

# 关闭所有LED（可选）
for led in leds:
    led.value(0)

# 主循环：检测按键并控制LED
while True:
    for i in range(4):
        # 按键按下时值为0
        if buttons[i].value() == 0:
            # 消抖处理：检测到按下后延时再确认
            time.sleep_ms(20)
            if buttons[i].value() == 0:
                leds[i].value(1)  # 点亮LED（高电平有效）
        else:
            leds[i].value(0)      # 熄灭LED
    time.sleep_ms(50)  # 主循环延时，降低CPU占用
