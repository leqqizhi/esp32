  • 红外避障传感器介绍

该传感器模块对环境光适应能力强，其具有一对红外线发射与接收管，发射管发射出一定频率的红外线，当检测方向遇到障碍物（反射面）时，红外线反射回来被接收管接收，经过比较器电路处理之后，蓝色指示灯会亮起，同时信号输出接口输出数字信号（一个低电平信号），可通过电位器旋钮调节检测距离，有效距离范围 2~30cm，工作电压为 3.3V-5V。该传感器的探测距离可以通过电位器调节、具有干扰小、便于装配、使用方便等特点，可以广泛应用于机器人避障、避障小车、流水线计数及黑白线循迹等众多场合。

黑色会吸收红外光，所以一般认为黑色为无障碍的情况，一般用来做小车的巡线引导

  • 实物展示

  1. 我们使用的这款红外避震传感器是可以进行数字信号和模拟信号切换，D为数字信号，A为模拟信号（如上图）

  2. 蓝色的方块上有个十字孔是用来调节灵敏度的。使用前，我们可以调节到蓝色指示灯刚好熄灭的状态。

实验一：检测信号数值

  • 信号检测

  • 数字信号检测

```python ''' 红外避障传感器信号检测（数字信号） ''' from machine import Pin,ADC import time

# 在 2 引脚创建 Pin 对象 infrared = Pin(2, Pin.IN) while True: print(infrared.value()) time.sleep(0.1) ```

测试结果：没有障碍物，值显示为“1”，有障碍物，返回值为"0"

  1. 模拟信号检测

```python ''' 声音传感器信号检测（模拟） ''' from machine import Pin,ADC import time

# 在 32 引脚创建 ADC 对象 infrared = ADC(Pin(2, Pin.IN),atten=ADC.ATTN_11DB)

while True: print(infrared.read_u16()) time.sleep(0.1) ```

测试结果：1. 没有障碍物时的数值较大，30000以上

2. 有障碍物时候数值降低 3000以下

实验二：障碍物报警器

  • 数字信号程序如下
'''
障碍物报警器（数字信号）
当有障碍物靠近时候灯亮，障碍物离开灯灭
'''
from machine import Pin
import time

infrared = Pin(2, Pin.IN) # 初始化红外避障传感器2号管脚
led = Pin(4, Pin.OUT)  # 初始化LED灯4号管脚

while True:
    if not infrared.value():
        led.value(1)
    else :
        led.value(0)
  • 模拟信号程序如下
'''
障碍物报警器（模拟信号）
当有障碍物靠近时候灯亮，障碍物离开灯灭
'''
from machine import Pin,ADC
import time


infrared = ADC(Pin(2, Pin.IN),atten=ADC.ATTN_11DB) # 初始化红外避障传感器2号管脚
led = Pin(4, Pin.OUT)  # 初始化LED灯4号管脚

while True:
    if infrared.read_u16() < 30000:
        led.value(1)
    else :
        led.value(0)
