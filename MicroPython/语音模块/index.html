  • SU-03T语音识别控制模块介绍

SU-03T语音识别控制模块为用户提供便捷便宜的串口离线语音识别方案，可广泛且快速应用于智能家居，各类智能小家电，86盒，玩具，灯具等需要语音操控的产品。 语音识别采用了离线语音识别芯片。通过串口通信的方式实现与主机的交互，最多支持150条语音指令，可在用户平台上自主定制语音指令和固件，命令词可任意修改，开发简单，不需要写代码。自带固件下载芯片，只需数据线即可完成固件下载。支持语音调节音量、学习唤醒词，识别灵敏且准确率高。

实验一：语音控制LED灯的亮灭

代码如下：

import machine
import time

uart1=machine.UART(1, tx=17, rx=16, baudrate=9600)
pin2 = machine.Pin(2, machine.Pin.OUT)

while True:
    if uart1.any():
        data = uart1.read()
        print(data)
        if data == b'\x01':  # 这是语音模块传回来的数据值01
            pin2.value(1)
            time.sleep_ms(100)
        elif data == b'\x02':  # 这是语音模块传回来的数据值02
            pin2.value(0)
            time.sleep_ms(100)

实验二：语音语音播报实时时间是温度

代码如下

'''
实时播报时间和温度
唤醒词：你好小美
询问时间：现在的时间是
询问温度：现在的是温度是
'''
from machine import UART, RTC,Pin
import time
import onewire
import ds18x20

# ===========初始化语音模块=UART1========
uart = UART(1, baudrate=9600, tx=17, rx=16)

# ======== DS18B20温度传感器配置 ========
ow = onewire.OneWire(Pin(4, Pin.IN, Pin.PULL_UP))  # 构建单总线对象
ds = ds18x20.DS18X20(ow)    # 构建DS18B20对象
roms = ds.scan()  # 扫描设备

# ========获取当前的温度值===============
def get_temp_hex():
    ds.convert_temp()  # 启动温度转换
    time.sleep_ms(750)  # 等待温度转换完成
    temp = int(ds.read_temp(roms[0])*100)  # 读取温度值
    temp_zs = temp // 100  # 温度的整数位
    temp_xs = (temp % 100)//10
    temp_xg = (temp % 100)%10
    return [temp_zs,temp_xs,temp_xg]

# ======获取当前时间值=============
def get_time_hex():
    # 初始化RTC
    rtc = RTC()
    # 获取当前时间 (年,月,日,星期,时,分,秒,微秒)
    now = rtc.datetime()
    # 转换为十六进制数组 [时,分,秒]
    return [now[4], now[5], now[6]]  # 时,分,秒


while True :
    # =======温度数据======
    temp_data = get_temp_hex()
    print(f'当前的温度数据为{temp_data}')

    # =======时间数据======
    time_data = get_time_hex()
    print(f'当前的时间数据为{time_data}')

    # =====发送时间十六进制信号======
    hex_time_data = [0xAA,0x55] + [0x02] + time_data + [0x55,0xAA]  # 添加帧头帧尾 (示例: 0xAA开头, 0x55结尾)
    byte_time_data = bytearray(hex_time_data)
    uart.write(byte_time_data)

    # =====发送温度十六进制信号======
    hex_temp_data = [0xAA,0x55] + [0x01] + temp_data + [0x55,0xAA] # 添加帧头帧尾 (示例: 0xAA开头, 0x55结尾)
    byte_temp_data = bytearray(hex_temp_data)
    uart.write(byte_temp_data)

    time.sleep(6)
