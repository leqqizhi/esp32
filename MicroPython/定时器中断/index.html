定时器的类和函数讲解

ESP32单片机定时器的功能在设定的时间后发生中断，触发执行指定的程序，定时器使用machine模块中Timer类及其内部函数：

  1. 构造定时器对象

在使用定时器前，先要通过Timer类构件定时器对象，语法如下：

machine.Timer(id)

*id*为定时器的编号，编号为0~3，目前ESP32仅支持同时4个定制器，当编号大于3时，会对编号进行取余，并覆盖之前相同编号的定时器。

  1. 定时器的函数体

定时器对象.init(period, mode,callvback)

参数 类型 必填 说明
‌id‌ int ✓ 定时器编号（不同硬件支持范围不同，通常从0开始）
‌mode‌ Timer.常量 ✕ 定时器模式：PERIODIC（周期）或 ONE_SHOT（单次），默认周期
‌period‌ int ✕ 周期时长（毫秒）
‌callback‌ function ✕ 中断回调函数，格式：callback(timer)

实验一：使用中断实现1个LED灯的闪烁

示例：

from machine import Timer, Pin
import time

led = Pin(2, Pin.OUT)  # ESP32 板载 LED（GPIO2）

# 定义回调函数
def toggle_led(timer):
    print("触发定时器0.5秒一次")
    led.value(not led.value())  # 翻转 LED 状态 

# 初始化定时器（周期 500ms）
tim = Timer(0)
tim.init(period=500, mode=Timer.PERIODIC, callback=toggle_led)

# 主循环可执行其他任务
while True:
    print("1秒一次")
    time.sleep(1)


实验二：使用定时器实现4盏LED灯的不同频率的闪烁

程序如下：

from machine import Timer, Pin
import time

led1 = Pin(2, Pin.OUT)  
led2 = Pin(4, Pin.OUT)
led3 = Pin(12, Pin.OUT)
led4 = Pin(13, Pin.OUT)

# 定义回调函数
def toggle_led1(timer):
    print("触发定时器0.25秒一次")
    led1.value(not led1.value())  # 翻转 LED 状态

def toggle_led2(timer):
    print("触发定时器0.5秒一次")
    led2.value(not led2.value())  # 翻转 LED 状态

def toggle_led3(timer):
    print("触发定时器0.75秒一次")
    led3.value(not led3.value())  # 翻转 LED 状态

def toggle_led4(timer):
    print("触发定时器1秒一次")
    led4.value(not led4.value())  # 翻转 LED 状态



# 初始化定时器（周期 500ms）
tim1 = Timer(0)
tim1.init(period=250, mode=Timer.PERIODIC, callback=toggle_led1)

tim2 = Timer(1)
tim2.init(period=500, mode=Timer.PERIODIC, callback=toggle_led2)

tim3 = Timer(2)
tim3.init(period=750, mode=Timer.PERIODIC, callback=toggle_led3)

tim4 = Timer(3)
tim4.init(period=1000, mode=Timer.PERIODIC, callback=toggle_led4)

# 主循环可执行其他任务
while True:
    pass

实验三：对实验二程序进行简化

程序如下：

from machine import Pin,ADC,PWM
import time

# 初始化引脚
encoder = ADC(Pin(32))  # 旋钮接32号引脚
# 将ADC衰减量设置为11DB，转换成0~3.6v范围的模拟电压
encoder.atten(ADC.ATTN_11DB)
motor_a = PWM(Pin(17), freq=1000)      # 电机引脚A
motor_b = PWM(Pin(16), freq=1000)      # 电机引脚B

value = 0 # 初始值为0
speed = 0 # 初始速度为0

# 定义一个映射函数        
def math_map(x, in_min, in_max, out_min, out_max):
    """
    将 x 从 [in_min, in_max] 线性映射到 [out_min, out_max]
    """
    return (x - in_min) * (out_max - out_min) / (in_max - in_min) + out_min

while True:
    value = encoder.read()  # 值的范围为0~4095
    speed = int(math_map(value,0,4095,0,1023))
    print(speed)
    motor_a.duty(speed)
    motor_b.duty(0)
