  • 水位传感器介绍

水位传感器专为水深检测而设计，可广泛用于感应降雨，水位，甚至液体泄漏。当将水位传感器放入水中时，水位没过铜线越多模拟值越大。读取水深传感器模块的模拟值，在串口打印出来。如需知道水深，我们直接将传感器放探测部分前面的一点点放入水中，这样我们就可以知道刚放入水中时的模拟值，知道这个值之后我们可以得出一个公式： 当前模拟值为CA ，放入水下时的模拟值为DA 。水深 =（CA-DA）/100。

实验一：检测信号数值

  • 信号检测

程序如下

```python ''' 水位传感器信号检测（模拟模式） ''' from machine import Pin, ADC import time

# 在 4 引脚创建 Pin 对象 level = ADC(Pin(4, Pin.IN),atten = ADC.ATTN_11DB) while True: print(level.read_u16()) time.sleep(0.1) ```

测试结果：1. 水位正常时，数值偏小

2. 水位的增加数值变大，（大于40000时，视为超过警戒水位）

实验二：水位报警器

  • 程序如下
'''
水位报警器
水位高于警戒水位是发出警报声
'''
from machine import Pin, ADC
import time

# 初始化有源蜂鸣器
buzzer = Pin(2, Pin.OUT)
# 在 4 引脚创建 Pin 对象
level = ADC(Pin(4, Pin.IN),atten = ADC.ATTN_11DB)
while True:
    if level.read_u16() > 40000:
        buzzer.value(1)
        time.sleep(0.2)
        buzzer.value(0)
        time.sleep(0.2)
