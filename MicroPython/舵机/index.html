舵机(Servo Motor)介绍

舵机(Servo Motor)是一种带有反馈控制系统的微型伺服电机，能够精确控制转动角度。它集成了电机、减速齿轮组、控制电路和位置反馈装置，可以根据控制信号精确地转动并保持在指定角度。

  • 舵机的工作原理接收PWM(脉宽调制)信号（这里以180°常见舵机为例）

  • 标准周期：20ms(50Hz)

  • 脉宽范围：通常500μs(0.5ms)到2500μs(2.5ms)对应角度：脉宽与角度成线性关系(如500μs=0°，2500μs=180°)

  • 其他角度舵机对应脉冲关系计算公式：

    $\text{脉冲宽度} = 500 + \left( \text{角度} \times \frac{2000}{\text{最大角度}} \right)$

示例代码：

from machine import PWM, Pin
import time

# 初始化PWM（GPIO15，50Hz）
servo = PWM(Pin(15), freq=50)

def set_angle(angle, max_angle=180):
    # 将角度转换为脉宽（适用于180°或270°舵机）
    us = 500 + (angle * 2000 // max_angle)
    duty = us * 1024 * 50 // 1000000  # 转换为duty值
    servo.duty(duty)

# 180°舵机转到90°
set_angle(90, max_angle=180)


舵机的转动的角度是通过调节 PWM（脉冲宽度调制）信号的占空比来实现的，标准 PWM（脉冲宽度调制）信号的周期固定为 20ms（50Hz），理论上脉宽分布应在 1ms 到 2ms 之间，但是，事实上脉宽可由 0.5ms 到 2.5ms 之间，脉宽和舵机的转角 0°～180° 相对应。

0.5-2.5ms 的 PWM 高电平部分对应控制 180 度舵机的 0-180 度，因此，对应的控制关系是这样的：

脉宽（μs） 180°舵机 270°舵机 说明
500 0° 0° 最小位置
1000 45° 67.5°
1500 90° 135° 中位（默认位置）
2000 135° 202.5°
2500 180° 270° 最大位置

实验一：舵机的来回转动

第三方库的使用

MicroPython 拥有着庞大的用户群，自然舵机模块也有开源的代码，直接拿过来使用即可，这就是使用 MicroPython 开发的高效之处，市面上常见的模块在网上几乎都可以找到相应的模块代码，大家一定要善于在网上搜索资源。

在项目文件夹中新建一个文件夹libs（名字自己取，要求是英文）用来放置我第三方库文件，然后在文件夹中，新建一个Python文件，我们示例为：Servo

  • 程序如下：

from machine import Pin, PWM
import time

class Servo:
    """
    MicroPython 舵机控制类
    适用于标准PWM舵机(周期20ms, 脉宽0.5ms-2.5ms)
    """

    def __init__(self, pin, min_us=500, max_us=2500, angle=180):
        """
        初始化舵机
        :param pin: 控制引脚
        :param min_us: 最小脉宽(微秒)，默认500us(0度)
        :param max_us: 最大脉宽(微秒)，默认2500us(180度)
        :param angle: 舵机转动范围，默认180度
        """
        self.pwm = PWM(Pin(pin))
        self.pwm.freq(50)  # 标准舵机频率50Hz(周期20ms)
        self.min_us = min_us
        self.max_us = max_us
        self.angle = angle
        self.current_angle = 0  # 记录当前角度

    def write_us(self, us):
        """直接写入脉宽(微秒)"""
        us = min(self.max_us, max(self.min_us, us))
        duty = us * 1024 * self.pwm.freq() // 1000000
        self.pwm.duty(int(duty))

    def write_angle(self, degrees):
        """写入角度"""
        degrees = min(self.angle, max(0, degrees))
        self.current_angle = degrees
        us = self.min_us + (self.max_us - self.min_us) * degrees // self.angle
        self.write_us(us)

    def sweep(self, start_angle=0, end_angle=180, step=1, delay=0.01):
        """舵机扫动"""
        step = abs(step) if start_angle < end_angle else -abs(step)
        for angle in range(start_angle, end_angle + step, step):
            self.write_angle(angle)
            time.sleep(delay)

    def off(self):
        """关闭PWM输出(释放舵机)"""
        self.pwm.duty(0)

    def deinit(self):
        """释放资源"""
        self.off()
        self.pwm.deinit()
  • 接下来在主程序中导入刚才的库文件就可以使用了
# 导入刚才的第三方库文件
from libs.Servo import Servo
import time

# 通过类实例化一个对象，传入参数为对应的管脚号，如果是其他角度的舵机，可以修改对应的angle参数值
serv0 = Servo(pin=4,angle=270)

# 调用类里面的write_angle（）直接输入角度，就可以执行转动到对应的角度
serv0.write_angle(180) # 写入角度
time.sleep(1)

# 重新回到0度
serv0.write_angle(0)
time.sleep(1)

# 左右全角度扫动3次
for i in range(3):
    for i in range(271):
        serv0.write_angle(i)
        time.sleep(0.01)
    for i in range(270,0,-1):
        serv0.write_angle(i)
        time.sleep(0.01)
